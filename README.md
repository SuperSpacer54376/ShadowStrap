# Shadowstrap Roblox Bootstrapper

![Shadowstrap Logo](logo.png)

Shadowstrap is a fork of Froststrap/Fishtrap/Bloxstrap designed as a powerful alternative bootstrapper for Roblox. It provides an enhanced Roblox launch experience with extra features, customization, and performance improvements.

## Features
- Drop-in replacement for the standard Roblox bootstrapper
- Customizable launcher appearance including themes and branding
- FastFlag editing and advanced Roblox client management
- Persistent client modifications through updates
- Integration support for Discord Rich Presence without security risks
- Multi-instance launching and server region display
- Self-updating mechanism ensuring you always have the latest bootstrapper
- Smooth handling of Roblox installations and updates

## Installation
1. Download the latest Shadowstrap release from the GitHub Releases page.
2. Run the installer and follow prompts to install Shadowstrap.
3. Launch Roblox through Shadowstrap for enhanced features and control.
